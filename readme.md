## ASCII Search

Search for ASCII characters by name or by number.

## Installing

Add `tinytacoteam/ascii-calculator` inside of `plugins` block of your  `~/.zazurc.js` file.

~~~ javascript
module.exports = {
  // ...
  'plugins': [
    'tinytacoteam/ascii-calculator',
    // ...
  ],
  // ...
}
~~~
